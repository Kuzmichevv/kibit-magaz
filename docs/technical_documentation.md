#Призначення системи

`KIBIT Magaz` — це вебзастосунок інтернет-магазину, який дозволяє користувачу переглядати каталог товарів, додавати товари до кошика, змінювати їх кількість і оформляти замовлення.

Архітектура

Проєкт побудований за схемою клієнт-сервер:

- клієнтська частина: HTML-шаблони Jinja2 та CSS;
- серверна частина: Flask-застосунок у `app.py`;
- стан кошика: зберігається в `session`.

Основні маршрути

- `GET /` — головна сторінка з каталогом товарів;
- `POST /add/<product_id>` — додавання товару до кошика;
- `GET /cart` — перегляд кошика;
- `POST /update/<product_id>` — зміна кількості товару;
- `POST /checkout` — оформлення замовлення.

Основні структури даних

`PRODUCTS`

Список словників, де кожен словник описує товар:

- `id` — унікальний ідентифікатор товару;
- `name` — назва товару;
- `price` — ціна у гривнях;
- `description` — опис товару;
- `category` — категорія товару.

`session["cart"]`

Словник виду:

```python
{
    "1": 2,
    "5": 1
}
```

Ключ — ідентифікатор товару у вигляді рядка, значення — кількість товару в кошику.

Залежності

- `Flask` — фреймворк для створення вебзастосунку;
- `Jinja2` — шаблонізатор для HTML;
- `Werkzeug` — обробка HTTP-запитів і службові функції Flask.

Тестування

Для перевірки працездатності реалізовано модульні тести, які покривають:

- відкриття головної сторінки;
- додавання товару до кошика;
- оновлення кількості;
- перевірку валідації форми замовлення;
- успішне оформлення замовлення.
