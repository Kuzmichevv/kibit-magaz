Інструкція користувача

Вимоги до середовища

- Windows;
- Python 3.14 або новіший;
- Visual Studio Code або інший редактор;
- доступ до терміналу PowerShell.

#Встановлення та запуск

1. Відкрити папку проєкту у Visual Studio Code.
2. Відкрити термінал.
3. Виконати команди:

powershell
py -m venv venv
.\venv\Scripts\python.exe -m pip install -r requirements.txt
.\venv\Scripts\python.exe app.py

4. Перейти в браузері за адресою `http://127.0.0.1:5000`.

Використання

1. На головній сторінці переглянути каталог товарів.
2. Натиснути кнопку додавання товару до кошика.
3. Відкрити сторінку кошика.
4. Змінити кількість товарів за потреби.
5. Ввести ім'я та адресу доставки.
6. Натиснути кнопку оформлення замовлення.

#Налаштування вмісту магазину

- щоб змінити товари і ціни, потрібно відредагувати список `PRODUCTS` у `app.py`;
- щоб змінити назву сайту, потрібно відредагувати `templates/base.html`;
- щоб змінити кольори сайту, потрібно відредагувати `static/style.css`.
