Інструкція користувача

Вимоги до середовища

Windows;
Python 3.14 або новіший;
Visual Studio Code або інший редактор;
доступ до терміналу PowerShell.
#Встановлення та запуск

Відкрити папку проєкту у Visual Studio Code.
Відкрити термінал.
Виконати команди:
py -m venv venv 
.\venv\Scripts\python.exe -m pip install -r requirements.txt 
.\venv\Scripts\python.exe app.py

Перейти в браузері за адресою http://127.0.0.1:5000.

На головній сторінці переглянути каталог товарів.
Можете обрати один з каталогів товару.
Натиснути кнопку додавання товару до кошика.
Відкрити сторінку кошика.
Змінити кількість товарів за потреби.
Ввести ім'я та адресу доставки.
Натиснути кнопку оформлення замовлення.

Налаштування вмісту магазину

щоб змінити товари і ціни, потрібно відредагувати список PRODUCTS у app.py;
щоб змінити назву сайту, потрібно відредагувати templates/base.html;
щоб змінити кольори сайту, потрібно відредагувати static/style.css;
щоб змінити сторінку категорій можна змінити у файлі templates/categories.html.
